# SOC Reporting Tool

This is a React application built with Vite and TypeScript. It is designed to provide a dashboard for SOC reporting, including charts (Recharts), Excel parsing (SheetJS), and icons (Lucide React).

## Setup

```bash
# install dependencies
npm install

# start development server
npm run dev
```

## Project Structure

- `src/components` – reusable UI components
- `src/pages` – top-level page components
- `src/utils` – utility functions (e.g. Excel parsing)

## Dependencies

- React
- Recharts
- xlsx (SheetJS)
- Lucide React

## Dev Dependencies

- Vite
- TypeScript
- @vitejs/plugin-react


> ⚠️ Node.js and npm must be installed on your system to run the project.
